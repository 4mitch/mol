# $mol_checker

A typical Checkbox element.

## [Online demo](http://eigenmethod.github.io/mol/#demo=mol_checker_demo)

## Usage example
```
...
childs /
	$mol_checker
		childs / \Unchecked
	$mol_checker
		childs / \Checked
		checked true	
```
## Properties

**`checked()`**

Property which state can be switched if a user clicked on element  

**`enabled()`**

Property that changes element's condition.
